### 1.项目背景及意义

****

目前的光学字符识别技术已经非常成熟，类似于有道词典的截图翻译也做得非常好，但是这类工具的定制化程度较低，无法做一些功能性的扩展，基于此问题，我结合现有的开源别的算法做一个简单的OCR识别程序。

### 2.项目内容

****

本项目使用python进行代码的编写，其中使用了ddddocr库作做识别，并使用wxpython做了一个简单的界面
